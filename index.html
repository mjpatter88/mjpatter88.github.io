Sound Processing

This project consisted of several different pieces, but they all were closely related to processing digital sound. The main objective of the overall project was to be able to generate sound files in a specific format and then do various types of processing on these sound files including using the Fourier Transform to determine the various frequency components making up the sound sample. This is similar to the way in which telephone exchanges work, so we also were required to generate the correct sound file for a given sequence of key presses. This sound file could be fed into our program that utilized the Fourier Transform and then analyzed to reproduce the original sequence of telephone key presses. Also, a final part to this project required that we write a GUI in Java which displayed multiple graphs relatd to the sound sample in a given input file. This GUI was required to use JNI to use the functionality we had developed in the C language in the other parts of the project.

This assignment was completed on Linux, so I learned how to use that operating system and the tool chain used to create C programs. I learned a great deal about file input and output and various data formats. This project also forced me to become extremely comfortable with pointers, memory allocation, and the inner workings of the C language. Not only that, but implementing the Fourier Transform in C also taught me a great deal about how to go from a mathematical formula to an algorithm and finally to working code.

In addition to all these technical details and many more, I also learned how to solve complex problems by breaking them down into manageable chunks. When faced with the full complexity of the program, it was daunting to even know how to begin. However, by taking an organized approach and intentionally making my code reusable, I was able to complete this project, meet all the requirements, and earn a perfect score of 1900/1900.

  • Date: Fall 2009
  • Technologies: C, Java, JNI, Make
